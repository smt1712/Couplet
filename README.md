## 项目名称
### Couplet
Tensorflow实现 seq2seq，并训练实现对对联

## 数据
70w+对的对联数据

## 主要功能
输入完整的上联，即可得到对应的下联

## 实现流程
- 数据预处理：构建字典
- 构造模型：构建Seq2Seq模型
- 训练模型：通过训练数据训练模型
- 测试模型：运行相关代码，测试模型的学习效果
- 启动服务器端：体验对对联的乐趣

## 运行语句
### 训练模型

    python train.py
### 启动服务器端

    python app_run.py 启动服务器端 浏览器输入127.0.0.1：5000即可体验
